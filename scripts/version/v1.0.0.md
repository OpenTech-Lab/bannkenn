## Install

**Linux x64:**
```bash
curl -Lo bannkenn-agent https://github.com/OpenTech-Lab/bannkenn/releases/download/v1.0.0/bannkenn-agent-linux-x64
chmod +x bannkenn-agent && sudo mv bannkenn-agent /usr/local/bin/
```

**Linux ARM64:**
```bash
curl -Lo bannkenn-agent https://github.com/OpenTech-Lab/bannkenn/releases/download/v1.0.0/bannkenn-agent-linux-arm64
```

```powershell

```
